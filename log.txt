## Ejercicio Avanzado: Generador de Sucesos Aleatorios con Restricciones

**Contexto:**
Imagina que estás desarrollando un pequeño juego de simulación donde los jugadores deben adivinar números generados aleatoriamente según ciertas reglas. Cada número generado debe cumplir con restricciones específicas y el proceso de generación debe ser repetido hasta que se cumplan todas las condiciones.

**Enunciado:**
Escribe un programa en Python que genere un número entero entre 1 y 100 (inclusive) utilizando un bucle `while`. El número generado debe cumplir con las siguientes restricciones:

1. **Suma de los dígitos**: La suma de los dígitos del número debe ser mayor a 8.
2. **Multiplicidad relativa**: Los dígitos del número no deben compartir factores primos entre sí, excepto el 1 (es decir, deben ser coprimos).

El programa debe repetir el proceso de generación hasta que se encuentre un número que cumpla ambas restricciones. Luego, debe imprimir el número encontrado y la cantidad de intentos realizados para generar ese número.

**Consideraciones:**
- Utiliza únicamente los conceptos proporcionados: expresión booleana, expresión de comparación, tipo de dato, algoritmo, identificador, flujo de ejecución, expresión, variable, operador, indentación, tipo de dato básico, literal, programa y bucle `while`.
- No utilices funciones adicionales ni módulos externos como `random` o `math`. Para generar un número aleatorio entre 1 y 100, puedes utilizar el siguiente fragmento de código:

```python
def generar_numero_aleatorio(seed):
    seed = (seed * 9301 + 49297) % 233280
    return seed % 100 + 1
```
Este código genera un número pseudoaleatorio entre 1 y 100 basado en una semilla proporcionada.

**Ejemplo de Salida:**
```
Número generado: 74
Cantidad de intentos: 12
```

**Nota:**
Asegúrate de que tu programa funciona correctamente para cualquier secuencia de números generados por la función `generar_numero_aleatorio`. El número de intentos puede variar dependiendo de la secuencia aleatoria obtenida.
